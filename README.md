# Wizard

Gestor de finanzas personales en un solo archivo HTML. Sin servidor, sin cuentas: todos los datos se guardan localmente en tu navegador (`localStorage`). Funciona offline e instalable como app (PWA).

## Uso

Abre `index.html` en cualquier navegador moderno, o visita la versión publicada.

### Instalar como app

- **iPhone/iPad (Safari):** Compartir → Agregar a pantalla de inicio
- **Android (Chrome):** Menú ⋮ → Instalar aplicación
- **Mac/PC (Chrome/Edge):** ícono de instalar en la barra de direcciones

## Respaldos

Tus datos viven solo en este dispositivo. Usa **Más → Exportar copia de seguridad** periódicamente para descargar un `.json`, y **Importar** para restaurarlo en otro equipo o tras limpiar el navegador.

## Desarrollo

Todo está en `index.html` (HTML + CSS + JS embebidos). No hay build step.
